# Notes to Booking App


## Commands and steps
1. 
```js
npm init -y
npm add express
```
adding dir app
adding file index.js => adding express + adding script in package.json
```json
{
    "scripts": {
        "start": "node index.js"
    }
}
```
2.
```js
npm add nodemone
//to watch changes for backend
```
```json
{
    "scripts": {
        "start": "nodemon index.js" //change
    }
}

3. MongoDb
- go to mongoDb (cloud mongodb)
- creating account, cluster, connect with account

```js
npm install mongodb
```
- creating .env and paste link form mongoDb with password of the user and (ip address 0.0.0.0, server adres after deploy)
